Branches of Algebra

Elementary Algebra

Elementary algebra focuses on solving linear equations and inequalities, graphing lines, and basic functions. Key concepts include:

  • Variables and expressions
  • Linear equations and inequalities
  • Graphing lines
  • Basic functions (domain, range, composition)

Intermediate Algebra

Intermediate algebra builds upon elementary algebra and covers:

  • Quadratic equations and formulas
  • Systems of linear equations
  • Functions (domain, range, composition, inverse)
  • Graphing (lines, parabolas, circles)

Advanced Algebra

Advanced algebra explores more complex topics, including:

  • Polynomial equations and rational expressions
  • Complex numbers and algebraic properties
  • Advanced graphing (conic sections, polar coordinates)

Key Concepts

Variables and Expressions

Variables are symbols used to represent unknown or changing values, while expressions are combinations of variables, numbers, and operations.

Linear Equations and Inequalities

Linear equations involve a single variable and can be solved using addition, subtraction, multiplication, or division. Inequalities involve a variable and a comparison operator (e.g. <, >, etc.).

Quadratic Equations

Quadratic equations involve a variable raised to the second power and can be solved using factoring, completing the square, or the quadratic formula.

Functions

Functions relate inputs (domain) to outputs (range). Key concepts include:

  • Domain: the set of input values
  • Range: the set of output values
  • Composition: combining functions
  • Inverse: reversing the function's operations

Graphing

Graphing involves plotting points and drawing lines or curves to represent mathematical relationships. Key graphs include:

  • Lines: linear equations, slope, intercept
  • Parabolas: vertex, axis of symmetry
  • Circles: equations, center, radius

Systems of Equations

Systems of equations involve multiple equations with multiple variables. Key concepts include:

  • Substitution: solving one equation for a variable and substituting into another equation
  • Elimination: adding or subtracting equations to eliminate a variable
  • Matrices: representing systems of equations using matrices
